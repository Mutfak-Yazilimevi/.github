## 🚀 Başlangıç

### Gereksinimler

- Node.js >= 20 _(varsa)_
- .NET SDK >= 8.0 _(varsa)_
- Docker _(opsiyonel)_

### Kurulum

```bash
# Depoyu klonlayın
git clone https://github.com/Mutfak-Yazilimevi/REPO_ADI.git
cd REPO_ADI

# Bağımlılıkları yükleyin
# npm install   (Node)
# dotnet restore (.NET)
```

### Çalıştırma

```bash
# npm start
# dotnet run
```
