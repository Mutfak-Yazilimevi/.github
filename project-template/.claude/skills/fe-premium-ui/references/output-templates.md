## 10. Acceptance Criteria Template

```markdown
## Acceptance Criteria

- The primary action is visible above the fold.
- The layout uses a consistent grid and spacing scale.
- Components include hover, focus, loading, disabled, empty, and error states where relevant.
- The visual system uses defined tokens for color, type, radius, border, and shadow.
- The page works on mobile, tablet, and desktop.
- Motion explains state and respects reduced motion.
- The result looks like a real product, not a generic template.
```
