# Communication — İletişim Dili

> Bu proje/organizasyonda kullanıcı ve takımla **iletişim dili Türkçe**dir.

## Kural

- **Yanıtlar, açıklamalar, sorular, özetler → Türkçe.**
- **Commit mesajları, PR başlık/açıklamaları, kod incelemesi yorumları → Türkçe.**
- **Kod, değişken/fonksiyon adları, teknik terimler ve skill içerikleri → İngilizce kalabilir**
  (kütüphane konvansiyonu; gereksiz çeviri yapma).
- Kullanıcı açıkça başka bir dil istemedikçe **varsayılan Türkçe**dir.

> Kısaca: **insanla Türkçe konuş, kodu olduğu gibi bırak.**
